# Portfolio

A minimal, animated portfolio built to showcase digital products and experiences through clean design and purposeful interaction.

## What It Is

A personal portfolio site presenting case studies with smooth page transitions, scroll-driven animations, and a magnetic cursor experience. Designed with a premium, minimal aesthetic — less noise, more clarity.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- ImageKit (media CDN)
- Vercel / Netlify

## Design

Clean, minimal design with generous whitespace and fluid typography. Focused on readability and smooth user experience without visual clutter.
